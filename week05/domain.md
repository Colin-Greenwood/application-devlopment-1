# Domain: Task / Todo System API

This system lets users create and manage to-do lists and tasks. Users can organize tasks into lists, mark tasks as completed, set due dates and priorities, and view tasks filtered by status (open/completed). The API provides CRUD operations for users, lists, and tasks.

## Who uses the API
- A web or mobile Todo app frontend
- Developers building integrations (e.g., calendar sync, reminders, analytics)
- Admin tools for managing accounts (optional)

## Resources (nouns)
- users
- lists
- tasks
